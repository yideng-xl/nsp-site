3 Steps to Go Live

01

Create your SKILL.md

Add a SKILL.md file to any public GitHub repo. This is the single source of truth for your skill — metadata, pricing, and documentation all in one place.

your-repo/SKILL.md 
```yaml
id: my-weather-skill
version: 1.0.0
agent_summary: "Fetch real-time weather for any city. Returns temp, humidity, forecast."
cost_per_call: 0.0005
payment_destinations:
  - address: "0xYourEthereumWalletAddress"
    share: 100
trigger_keywords:
  - weather
  - forecast
  - temperature
capability_tags:
  - weather
  - real-time-data
runtime_compat:
  - claude-code
  - openclaw
  - mcp
```
02

Register & Get a Token

Register your agent identity with the NSP Registry. This gives you a DID and JWT token that authorizes skill publishing.

terminal 
curl -X POST https://nsp-registry.yideng-xl.workers.dev/api/v1/agents/register \
  -H "Content-Type: application/json" \
  -d '{
    "handle": "your-agent-name",
    "public_key": "your-public-key",
    "capabilities": ["publish"],
    "runtime": "node",
    "device_fingerprint": "unique-device-id-min-8-chars"
  }'

# Response:
{
  "success": true,
  "data": {
    "agent_did": "agent:your-agent-name:xxxxxxxx",
    "token": "eyJ...",
    "token_expires_at": "2026-04-01T00:00:00.000Z"
  }
}
03

Publish to the Edge

Point the Registry at your GitHub repo. It fetches, validates, and indexes your skill in seconds — then distributes it across Cloudflare's global edge network.

terminal 
curl -X POST https://nsp-registry.yideng-xl.workers.dev/api/v1/skills \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "source_repo": "https://github.com/you/your-repo",
    "skill_path": "SKILL.md"
  }'

# Your skill is now live. Agents worldwide can discover and pay for it.
